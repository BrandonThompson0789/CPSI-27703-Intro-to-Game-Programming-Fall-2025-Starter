====================================
    GAME DISTRIBUTION PACKAGE
====================================

Thank you for downloading this game!

HOW TO RUN:
-----------
1. Make sure all files are extracted to the same folder
2. Double-click on "demo.exe" to start the game
3. Enjoy!

SYSTEM REQUIREMENTS:
-------------------
- Windows 10 or later
- DirectX compatible graphics card

INCLUDED FILES:
--------------
- demo.exe (main game executable)
- *.dll files (required libraries - keep these in the same folder!)
- assets/ folder (game data - DO NOT delete or move this folder!)

CONTROLS:
---------
Check the input configuration files in the assets folder for control options.
Default controls can be customized via the input_config.json file.

TROUBLESHOOTING:
---------------
- If the game doesn't start, make sure all DLL files are in the same folder as demo.exe
- If textures don't load, ensure the assets folder is in the same directory as demo.exe
- Make sure you have the latest Visual C++ Redistributable installed

NOTES:
------
This is a debug build of the game. Performance may vary.

Enjoy playing!
